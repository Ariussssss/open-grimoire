- polybar: pretty bar
- variety: background image update
- i3-gaps
** start
Logout, change to i3 before login
** shortcut
- ~$mod~
  - navigation
    - ~Enter~ new terminal
    - ~d~ drun, like spotlight
    - ~shift+q~  close
    - ~h/v~ split direction
    - ~shift+<number>~ move to group
  - config
    - ~shift+e~ quit window manager
    - ~shift+c~ reload config
