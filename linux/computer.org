* Process and Thread

** Process
避免阻塞, 规划调用资源和共享存储指针地址

私有部分:
- 地址空间
- 全局变量

** Thread
避免阻塞, 规划 CPU 运算分配, 区分执行线程和 IO 线程协同.

具有部分进程的线程, 又称更轻量级进程(lightweight process).
其实设计和进程相似, 每个线程有自己的内存空间, 堆栈.
CPU 快速切换, 在运算不密集的多线程中, 模拟达到并行的效果. 

部分硬件支持多线程, 纳秒级线程切换

私有部分:
- 程序计数器
- 寄存器
- 堆栈
- 状态
