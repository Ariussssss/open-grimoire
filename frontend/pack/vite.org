* vite
** Refers
   - https://cn.vitejs.dev

** 对比 snowpack
   - 生产构建
     - snowpack 默认将文件单独成构建模块, 允许对应不同的优化器(类似 loader)
     - vite 统一使用 Rollup 集成优化:
       - 多页面应用支持
       - 库模式(预览演示)
       - 分割 css 不需要 js 化
       - 预先异步加载, 不需要等待浏览器解析 esm 依赖
       - 预构建, 强缓存, 请求 hash 控制刷新, 加快冷启动时间
   - 支持 monorepo
   - 内置 css 预处理器和依赖 alias
   - vue vip 级别优化
